TransactionID

The identifying hash of a transaction.

Warning: XRP-API is early alpha software. API methods and data formats are likely to change frequently in ways that break backwards compatibility.

  • Type: String

  • Pattern: ^[0-9A-F]{64}$

  • Example: "E08D6E9754025BA2534A78707605E0601F03ACE063687A0CA1BDDACFCD1698C7"
